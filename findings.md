## 一、核心概念（Core Concepts）

| 概念 | 说明 |
|------|------|
| **U 形槽本体** | 由底板 + 两边墙组成的 U 型钢筋混凝土结构，用于下穿引道、排水边沟、机非分隔带等；刚度大、变形小、收坡支挡与防水效果好。 |
| **分节与断面** | 按里程将 U 型槽划分为 10–20 m 小节，以起止断面及分界断面为计算剖面；每节有独立几何参数（Hmin/Hmax、L、B、T、底宽等）。 |
| **计算模型** | 边墙按刚性挡墙/悬臂梁；底板按弹性地基梁（文克尔或郭氏表法/链杆法）。取线路方向 1 延米简化分析。 |
| **控制截面** | 边墙与底板固结的 A、B 节点；底板悬臂段根部、固结段根部、跨中。上述位置弯矩最大且易应力集中。 |
| **配筋体系** | 受力主筋 + 箍筋 + 分布筋 + 控制截面加强筋；分单层/双层配筋；N1–N8 等编号区分部位与层位，内外侧可交错布置。 |
| **类型变体** | 主 U 形槽（YU 系列）、机非分隔带（Y01–Y19）、排水沟+盖板、防撞侧石等；存在普通/加厚断面、有孔盖板等。 |

## 二、设计规则（Design Rules）

### 2.1 几何与参数
- **断面参数**：L（长）、a/b、B（宽）、m、e1/e2、Hmin/Hmax；底宽、边坡、侧墙厚 B、底板厚 T。
- **平面图**：16 关键点由 wside（线边距）、win（线间距）、btop/bbot/btoe（边墙顶宽/底宽/趾板宽）、θ（线位角）及分界点坐标计算。
- **纵断面**：6 关键点由里程、边墙顶高程、节长、底板厚、边墙高确定。
- **横断面**：12 关键点，以中桩为基点，结合地面高程、边墙高、顶宽/底宽/趾板宽、底板厚。

### 2.2 荷载与组合
- **荷载**：结构自重、活载（换算土柱）、地基反力、地下水压力（静止）、边墙土压力（静止或主动+1.1–1.3 安全系数）。
- **抗浮**：Fs = γ(ΣGk+ΣGpk)/(Ffk+Pwy)，运营阶段 Fs≥1.0（有的取 ≥1.1）；浮力按水位以下面积及折减系数计算；可换填碎石垫层以排水、减水压、提高承载力。

### 2.3 材料与构造
- **混凝土**：垫层 C15；主体常用 C35/C40；防水时 C40、P8，迎水面全包附加防水层。
- **钢筋**：HRB400（或 HRB335）主筋；HPB300 箍筋/构造筋；常见直径 φ8、φ10、φ12、φ16、φ18、φ20、φ22。
- **保护层**：通常 4–5 cm（未注按 4.0 cm）；排水沟等小构件 3 cm；防水设计对迎水面/背水面保护层有专门要求。
- **锚固与搭接**：锚固长度 65d（或按图）；搭接 20d；弯钩搭接 ≥10 cm；弯折直径 ≥3 cm；弯钩未注均为标准弯钩。

### 2.4 配筋与加强
- **边墙**：按悬臂梁正截面受弯、斜截面受剪配筋；满足裂缝宽度及构造要求；主筋间距满足最小间距。
- **底板**：按弹性地基梁弯矩、剪力配筋；除受力筋外设箍筋、分布筋、控制截面加强筋。
- **应力集中**：A、B 节点及悬臂/固结根部、跨中加密箍筋或设加强筋。
- **双层配筋**：N1–N8 等为双层时，内外侧交错；N1/N2 双层每层相同；N3/N4 单层时见配筋图。

### 2.5 防水与细部
- **原则**：以防为主、刚柔结合、多道防线；主体防水混凝土 + 迎水面附加防水层；伸缩缝/沉降缝/变形缝专项防水。
- **施工缝**：水平缝不宜留在底板与边墙交接处，宜高出底板 ≥300 mm；竖向缝宜与变形缝同断面。
- **伸缩缝**：设置 3 cm 等构造；止水带 + 防水材料 + 加强层；缝两侧连续、平顺。

### 2.6 地基与排水
- **地基**：承载力不满足时可采用换填、水泥搅拌桩、CFG 桩等；深厚软土需工后沉降分析。
- **排水**：渗透系数 K、水力坡度 I；扬压力/排水计算；排水沟断面与盖板配套（含普通/加厚、有孔盖板）。

## 三、标准流程与出图（SOP）

### 3.1 计算与分节
1. 从 CAD 或线位数据提取 U 型槽起止里程、底板顶/边墙顶/地面/地层高程。
2. 按 10–20 m 分节，计算每节里程、几何尺寸（Hmin/Hmax、B、T、底宽等）。
3. 各断面依次：边墙高 → 土压力 → 结构内力（弯矩、剪力）→ 截面配筋。
4. 抗浮、地基承载力、工后沉降等逐一验算；排水/扬压力按 K、I 等计算。

### 3.2 配筋与表格
1. 按截面与弯矩做单面/双面配筋计算，生成配筋表：钢筋类型、单根长、根数、总长、每米重、总重。
2. 钢筋数量表按 YU 型（或 Y 型、U 型）分型，列编号、配筋形式、长度、根数、单根重、总重；汇总 HPB300、HRB400、C35/C40 混凝土。
3. 下料长度以 mm 计，尺寸以 cm 计；列出大样与规格（如 φ22、φ18）。

### 3.3 出图
1. **平面图**：按每节里程与关键点公式绘制各节平面，再拼成整体 U 型槽平面图。
2. **纵断面图**：按边墙顶高程、每节里程与尺寸绘整体纵断面。
3. **横断面图**：以地质横断面中桩为基点，按地面高程与每节尺寸绘各横断面。
4. **配筋图**：纵向剖面、平面（A-A）、侧立面（B-B、C-C等）；标注 N×@间距、保护层、垫层、混凝土强度、设计水位等；未注尺寸见说明。

### 3.4 说明与图例
- 说明中明确：保护层厚度（未注取 4.0 cm）、弯钩标准、锚固/搭接规则、混凝土与钢筋等级。
- 附注中明确：双层配筋与交错规则、施工顺序（加工成型后现场绑扎）、与平面布置图/说明的衔接。
- 参照标准排水沟盖板图、标准图集等；伸缩缝、变形缝做法按项目要求。

## 四、表格设计逻辑（Spreadsheet Design Logic）

通过对 `DesignSheet` 中 Excel 表格（如“振华路”系列）的深度解析，提取出以下关键计算逻辑与自动化规则：

### 4.1 几何参数自动化
- **边墙厚度（B）自动查找**：基于最大计算高度（Hmax），通过 `LOOKUP` 函数自动匹配厚度：
  - `LOOKUP(MAX(J, I), {0, 3, 4, 5, 6.5, 7.5, 9, 10}, {0.3, 0.4, 0.5, 0.6, 0.7, 0.8, 0.9})`
  - *含义*：当 Hmax ≤ 3m 时 B=0.3m；3.5~4m 时 B=0.4m；以此类推，最高 10m 时 B=0.9m。
- **结构跨度/底宽**：由线位间距（win）及边墙厚度（B）自动生成：`Width = win + B * 2`。

### 4.2 荷载与验证逻辑
- **荷载分项系数**：依据公路等级（二级及以下 vs 其他）通过 `IF` 判断选择 0.95, 1.0 或 1.05。
- **抗浮验算（Fs）**：
  - 计算各节段的总重与静水压力比值。
  - 公式：`Fs = (自重 + 压重) / 浮力`，逻辑输出：`IF(Fs > 目标值, "YES", "NO")`。
- **截面验算逻辑**：
  - **最小配筋率**：`IF(ρ < 0.006, 0.006, IF(ρ > 0.02, 0.02, ρ))`。强制限制配筋率在 0.6%~2.0% 之间。
  - **主筋验算**：引用 `0.结构参数` 里的跨径与厚度，结合弯矩进行截面强度校核。
  - **Spacing Selection**: `LOOKUP` or `IF` used to choose spacing (s1, s3) based on calculated area `As`. Typical values: 100, 125, 150, 200.

### 4.3 排水与扬压力
- **降水井/渗流计算**：采用“无压非完整井”模型。
- **参数关联**：计算公式 `J - J_next`（如 `coord K7 = J5 - J7`）用于计算地层层厚，进而通过渗透系数（K）计算单井涌水量或总排水量。

### 4.4 Multi-Sheet Dependency Map
Based on `sheet_dependencies.json`, the data flow is as follows:
1.  **Level 0 (Base)**: `0.结构参数` (Structural Parameters), `0.地勘及桩计算` (Soil & Pile Specs).

### 4.5 Rebar Spacing & Rules (DrawingShot Refinement)
- **N1/N2 (Side Wall Main)**: Spacing typically @100mm or @125mm (s1).
- **N3/N4 (Bottom Plate Main)**: Spacing typically @100mm or @125mm (s3).
- **N5/N6 (Stirrups/Distribution)**: Spacing @200mm (s5/s6).
- **N7/N8 (Extra Reinforcement)**: Spacing @100mm (s1).
- **Corner Reinforcement**: A/B nodes (junction of side wall and bottom plate) often use @100mm spacing for N1/N2 extending into the plate.
- **Double Layer Cross-hatching**: Outer layer and inner layer rebar should be staggered/offset if specified.
