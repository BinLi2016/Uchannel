---
description: Transpile P-CAD code to AutoLISP and render it in AutoCAD.
---

This workflow automates the transpilation and rendering of P-CAD files.

1.  **Ask for P-CAD File**
    - Ask the user for the path to the `.pcad` file (e.g., `P-CAD-code\U-Bar.pcad`).

// turbo
2.  **Transpile to AutoLISP**
    - Run the transpiler command:
      ```powershell
      python f:\Uchannel\Python\pcad_to_autolisp.py <pcad_file_path> f:\Uchannel\P-CAD-code\AutoGenerated.lsp
      ```

3.  **Prepare AutoCAD Script**
    - Create a temporary script file `f:\Uchannel\P-CAD-code\render.scr` with the following content:
      ```lisp
      (load "f:/Uchannel/P-CAD-code/AutoGenerated.lsp")
      (c:PCAD_Render)
      (command "._ZOOM" "_ALL")
      ```

// turbo
4.  **Execute AutoCAD**
    - Launch AutoCAD and run the script:
      ```powershell
      & "C:\Program Files\Autodesk\AutoCAD 2021\acad.exe" /b "f:\Uchannel\P-CAD-code\render.scr"
      ```
    - *Note: If acad.exe is in a different location, the user or agent should adjust the path.*

5.  **Verify**
    - Confirm with the user that the drawing has been rendered correctly in AutoCAD.
