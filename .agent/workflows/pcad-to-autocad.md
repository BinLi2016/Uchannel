description: Transpile P-CAD code to AutoLISP and render it in AutoCAD.
---

This workflow automates the transpilation and rendering of P-CAD files.

1.  **Ask for P-CAD File**
    - Ask the user for the path to the `.pcad` file (e.g., `P-CAD-code\U-Bar.pcad`).

// turbo
2.  **Transpile to AutoLISP**
    - Run the transpiler command (this automatically generates `<filename>.scr` in ANSI):
      ```powershell
      python f:\Uchannel\.agent\skills\image-to-pcad\scripts\pcad_to_autolisp.py <pcad_file_path> f:\Uchannel\P-CAD-code\AutoGenerated.lsp
      ```

// turbo
3.  **Execute AutoCAD**
    - Drag and drop `f:\Uchannel\P-CAD-code\AutoGenerated.scr` into AutoCAD, or run:
      ```powershell
      & "C:\Program Files\Autodesk\AutoCAD 2021\acad.exe" /b "f:\Uchannel\P-CAD-code\AutoGenerated.scr"
      ```
    - *Note: If acad.exe is in a different location, the user or agent should adjust the path.*

5.  **Verify**
    - Confirm with the user that the drawing has been rendered correctly in AutoCAD.
