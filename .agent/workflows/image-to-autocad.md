---
description: Autonomously convert a rebar detail image to an AutoCAD drawing using P-CAD.
---

This workflow automates the conversion of rebar detail images into AutoCAD drawings using the `image-to-pcad` and `pcad-to-autocad` skills.

1.  **Ask for Image Path**
    - Ask the user for the absolute path of the rebar detail image (e.g., `f:\Uchannel\DrawingShot\Sidewall.jpg`).

2.  **Generate P-CAD Code**
    - Use the `image-to-pcad` skill (read `f:\Uchannel\.agent\skills\image-to-pcad\SKILL.md` for guidance).
    - Analyze the image and generate a `.pcad` file (e.g., `f:\Uchannel\P-CAD-code\AutoGenerated.pcad`).
    - Ensure logical parameter extraction and layer definitions.

3.  **Transpile and Render**
    - Use the `pcad-to-autocad` skill/workflow to complete the process.
    - // turbo
      ```powershell
      python f:\Uchannel\Python\pcad_to_autolisp.py f:\Uchannel\P-CAD-code\AutoGenerated.pcad f:\Uchannel\P-CAD-code\AutoGenerated.lsp
      ```
    - Create the AutoCAD script `f:\Uchannel\P-CAD-code\render.scr`:
      ```lisp
      (load "f:/Uchannel/P-CAD-code/AutoGenerated.lsp")
      (c:PCAD_Render)
      (command "._ZOOM" "_ALL")
      ```
    - // turbo
      ```powershell
      & "C:\Program Files\Autodesk\AutoCAD 2021\acad.exe" /b "f:\Uchannel\P-CAD-code\render.scr"
      ```

4.  **Verify Results**
    - Ensure the drawing in AutoCAD matches the original image's dimensions and reinforcement details.
